# DVWA Web Application Security Assessment

## Overview

This repository documents a Vulnerability Assessment and Penetration Testing (VAPT) performed against the Damn Vulnerable Web Application (DVWA).

The assessment was conducted inside a controlled laboratory environment using Docker and Kali Linux for educational and professional learning purposes.

---

## Disclaimer

This project was performed in an isolated lab environment on an intentionally vulnerable application.

No production systems or unauthorized targets were tested.

---

# Target

Damn Vulnerable Web Application (DVWA)

Environment:

- Kali Linux
- Docker
- Localhost

Target URL

http://127.0.0.1:9090

- Reconnaissance
- Technology Fingerprinting

- SQL Injection Testing

- Cross-Site Request Forgery (CSRF)

- Security Headers Analysis
- Automated Vulnerability Scanning
